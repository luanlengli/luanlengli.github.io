服务器配置

可以根据需要修改对应的配置，这里只做以下几项

  • 网络设置，配置网络，安装docker需要访问公网
    • 配置IP地址
    • 配置网关
    • 配置DNS

  • Windows 更新设置，安装docker前需要更新Windows
    • 修改为手动更新
    • 选择下载并安装更新
    • 在弹出命令窗口里面选择搜索所有的更新
    • 选择安装所有更新
    • 安装完成之后重启系统

手动安装Docker

  • 下载docker压缩包
  • 解压
  • 删除压缩包
  • 为当前会话添加PATH变量
  • 配置新的PATH变量
  • 将系统PATH变量替换为新的PATH变量
  • 注册为系统服务
  • 设置Docker开机启动
  • 启动Docker
